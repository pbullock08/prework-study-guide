HTML

  • Hypertext Markup Language (HTML) creates structure for a webpage.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • It is best practice to use semantic HTML elements in the body of the webpage to make the code easy to read and webpage accessible to all.
  • A tag refers to what appears inside the angle brackets, while an elment refers to what appears from opening to closing tag.

Git

  • Git is a system that lets you manage and keep track of each version of your project.
  • GitHub is a cloud-based hosting service that lets you keep track of your projects and collaborate easily.
  • Branches are features of the GitHub repository that allow you make changes in a contained area within the repository or project folder.
  • Typing git status into the command line checks what branch we are currently on.
  • Typing git checkout -b branch-name into the command line creates a new branch and switches to it.
